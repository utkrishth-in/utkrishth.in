SCP-API

This is an API that can fetch all information about all SCPS (ok maybe not the last few), but it's made in python, it first fetches data from the SCP wikidot and stores it in a github repo as json, then whenever it needs to be fetched, it fetches from the github repo, not the wikidot (inorder to prevent load).

You can use this with pip `pip install scpfetch`,
Documentation is available at Github
